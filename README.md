## Rodando localmente

Clone o projeto

```bash
  git clone git@github.com:marloncoutodev/portfolio.git
```

Entre no diretório do projeto

```bash
  cd portfolio
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run dev
```
